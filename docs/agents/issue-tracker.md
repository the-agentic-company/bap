# Issue Tracker: Linear

Issues for this repo are tracked in Linear under team `cmdlaw`.

## Agent workflow

Agents should prepare issue text only. Do not create or update Linear issues directly unless the user explicitly asks and Linear access is available.

When preparing an issue, include:

- A concise title
- The problem or requested change
- Relevant repo paths, commands, or observed behavior
- Acceptance criteria
- Suggested Linear label and status

Agents should choose appropriate Linear labels and statuses based on the issue content and the triage guidance in `docs/agents/triage-labels.md`.

Do not link issues to GitHub branches or pull requests unless the user explicitly asks.
